# PokemonTask
A modern Pokémon Explorer built with Angular Standalone Components, Angular Material, Signals, and PokéAPI.
# Features:
Search Pokémon by name
Fetch data from PokéAPI
Using Angular Signals
Angular Material UI (Select, Form Field)
Reusable standalone components
Clean and scalable architecture

# Tech Stack
Angular 17+ (Standalone)
Angular Material
RxJS
Signals
PokéAPI
